## 📄 Sobre o Projeto

Este projeto é uma página institucional da **Garbimaq**, empresa de Jundiaí (SP), que presta serviços técnicos em impressoras multifuncionais. A landing page foi desenvolvida com foco em:

- Responsividade
- Clareza nas informações
- Contato fácil via telefone, WhatsApp ou e-mail
- Navegação suave entre seções

---

## 🖥️ Tecnologias Utilizadas

- **HTML5**
- **CSS3**
- **JavaScript (Vanilla)**
- **Phosphor Icons** (ícones SVG via CDN)
- **Google Maps Embed**

---

## 🎯 Funcionalidades

- ✅ Menu lateral responsivo para dispositivos móveis
- ✅ Scroll suave até as seções do site
- ✅ Seção com destaque das **marcas atendidas**, com slideshow automático
- ✅ Links diretos para:
  - WhatsApp
  - Telefone
  - E-mail
  - Endereço no Google Maps
- ✅ Iframe com mapa de localização embutido
- ✅ Seção "Quem Somos", "Contato", "Endereço" e "Marcas Atendidas"
- ✅ Ícones modernos com Phosphor Icons
